bool UnifiedAutomation.UaServer.ChildInstanceSettings.OmitIfOptional
getset

Omits the instantiation of nodes with the optional modelling rule (ObjectIds.ModellingRule_Optional).

Usually, if there is a child instance settings instance for a node, the node is instantiated, no matter if the modelling rule is optional or mandatory. If OmitIfOptional is set to true, the child will only instantiated if the modelling rule is mandatory.
