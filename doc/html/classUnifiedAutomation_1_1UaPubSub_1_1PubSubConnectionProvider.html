Adds a profile to the connection provider.

The CreateConnection method will look up the suitable transport and encoding provider by means of the TransportProfileUri.

An application developer will usually use the convenience extension methods, like UdpExtensions.AddUdpUadpProfile(PubSubConnectionProvider). Those methods combine the correct set of profile uri, transport provider, and encoding provider.
