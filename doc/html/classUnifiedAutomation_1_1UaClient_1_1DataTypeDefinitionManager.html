Detailed Description

Manages access to the DataTypeDefinitions and DataTypeDictionaries exposed by a server.

Reads and evaluates DataTypeDefinitions on first use.

Member Enumeration Documentation

enum UnifiedAutomation.UaClient.DataTypeDefinitionManager.DataTypeDefinitionManagerConfiguration

Configuration whether DataTypeDefinitions or DataTypeDictionaries are evaluated.

Enumerator
Default 

Tries to read DataTypeDefinition attributes. If not present, try to read DataTypeDictionary

DataTypeDefinitionOnly 

Tries to read DataTypeDefinition attributes. No fallback to DataTypeDictionary.

DataTypeDictionaryOnly 

Tries to read the DataTypeDictionary. Shall only be used if the server has invalid DataTypeDefinition attributes.

enum UnifiedAutomation.UaClient.DataTypeDefinitionManager.ReconnectBehavior

Specifies the handling of the cached DataTypeDefinitions on reconnect.

Enumerator
ClearCacheOnNamespaceTableChange 

The cache gets cleared if the NamespaceTable changes

MapCachedIdsIfRequired 

If the NamespaceUris of the server changed on a reconnect (e.g. a server gets re-configured and restarted), the cached NodeIds will be mapped.

ClearCache 

The cache will get cleared.
