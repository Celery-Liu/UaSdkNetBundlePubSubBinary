Detailed Description

Reads objects from a XML stream.

In most cases, if no special communication settings have been made (see MessageContext), the default MessageContext is sufficient.

An example of loading and decoding a previously encoded EnpointDescription endpoint (see XmlEncoder).

MessageContext context = new MessageContext(_currentSession.Application.DefaultEndpointConfiguration);
document = new System.Xml.XmlDocument();
using (System.IO.TextReader tr = new System.IO.StreamReader("temp.xml", System.Text.Encoding.UTF8, false))
{
document.Load(tr);
}
XmlDecoder decoder = new XmlDecoder(new XmlString(document.InnerXml), context);
decoder.SetMappingTables(context.NamespaceUris, context.ServerUris);
decoder.PushNamespace(Namespaces.OpcUa);
EndpointDescription endpoint = (EndpointDescription)decoder.ReadEncodeable(nameof(EndpointDescription), typeof(EndpointDescription));
decoder.Close();
_currentSession.Connect(endpoint, null);
