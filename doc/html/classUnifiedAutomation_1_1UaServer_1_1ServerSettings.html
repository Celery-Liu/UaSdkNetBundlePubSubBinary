bool UnifiedAutomation.UaServer.ServerSettings.DisableRegisterServer2
getset

Specifies if the server should discard trying to register itself at the Local Discovery Server (lds) with the RegisterServer2 service but directly try to register with RegisterService. This setting is only evaluated if RegistrationEnabled is set to 'true'. The default value is 'false'

This setting should only be set to 'true' if an outdated lds is installed, that does not support the RegisterServer2 service.
