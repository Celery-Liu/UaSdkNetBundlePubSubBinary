out StatusCode  statusCode

Converts a sequence number of type ushort or uint to a ushort.

Parameters
sequenceNumber
statusCodeThe status code describing the conversion result. Possible values are: StatusCodes.Good, when the sequence number is representable as ushort, StatusCodes.GoodClamped when the original value is a 32-bit value that is not representable with a 16-bit value, or StatusCodes.BadNoData when the value is null.
Returns
The sequence number as ushort.

The DataSetMessage sequence number.

UADP is using ushort as data type, whereas JSON message mapping is using an uint. The producer of a DataSetMessage should always set an 32-bit unsigned integer value. The encoder will take care to convert it to 16-bit if needed.
