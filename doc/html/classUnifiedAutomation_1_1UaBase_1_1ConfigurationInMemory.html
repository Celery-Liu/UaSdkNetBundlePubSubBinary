void UnifiedAutomation.UaBase.ConfigurationInMemory.SetSecurity ( string  commonStorePath,
string  certificateSubjectName 
)

Configures the paths for the following stores:

  • ApplicationCertificate (/own)
  • TrustedCertificateStore (/trusted)
  • IssuerCertificateStore (/issuers)
  • RejectedCertificatesStore (/rejected)
Parameters
commonStorePathSpecified path for the base certificate store.
certificateSubjectNameThe specified SubjectName of the application instance certificate.

remarks> The specified SubjectName is used to find the certificate of the application at startup. E.g. "CN=GettingStartedServer/O=UnifiedAutomation/DC=localhost" /remarks>
