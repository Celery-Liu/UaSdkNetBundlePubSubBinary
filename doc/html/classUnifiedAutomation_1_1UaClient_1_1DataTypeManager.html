Detailed Description

An object which manages access to the data type dictionaries exposed by a server.

Structured DataTypes using structured DataTypes from other namespaces are not supported.

bool UnifiedAutomation.UaClient.DataTypeManager.DisableReadWithIndexRange
getset

If true the DataTypeDictionaries are read without IndexRange.

Set to 'true# if the server does not support reading with IndexRange. If 'true' reading the dictionaries could fail for big DataTypeDictionaries.
