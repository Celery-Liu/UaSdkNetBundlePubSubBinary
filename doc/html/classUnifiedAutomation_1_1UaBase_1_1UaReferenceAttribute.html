Detailed Description

Use to signal that a model instance has an additional reference.

The decorated property should be of the type NodeId, UaBase.ExpandedNodeId, IEnumerable{NodeId} or IEnumerable{ExpandedNodeId}. The value of the property represents the target node id or ids.
