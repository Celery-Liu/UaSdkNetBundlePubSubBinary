Detailed Description

https://reference.opcfoundation.org/v105/Core/docs/Part14/9.1.10/#9.1.10.1

The state of the PubSubStatusModel is influenced by several different actors, e.g., the logical parent, the communication back-end, the client calling methods etc. The implementation uses some substates to reflect the states and calculates from that the actual State. The substates directly affecting the state are ReducedState, ElementState and HasOperationalError. The ReducedState is itself a reduction of some substates, that are the IsParentEnabled, IsDisabled, and the IsConfiguring state.
